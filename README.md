## Key features
- CREATE - We can add data to database.
- READ - All the data from database is available in ALLUSERS page.
- UPDATE - Data of any user can be edited easily.
- DELETE - We can delete data of any user.

- Light and Dark mode
- Responsive
- Use of animations
